# 🚨 NCRB Crime Analytics & Data Warehouse Project (2022–2024)

An end-to-end Data Analytics & Business Intelligence project built using NCRB metro crime datasets from 2022–2024. The project covers complete data engineering workflow including raw data cleaning, preprocessing, SQL warehouse creation, analytical query generation, and interactive Tableau dashboard development.

---

# 📌 Project Overview

This project analyzes crime trends across major Indian metro cities using NCRB datasets. The workflow transforms raw Excel files into structured analytical datasets and visual dashboards for crime trend analysis and decision-making.

The project includes:

- Data Cleaning & Preprocessing using Python
- Structured Data Warehouse using SQL
- Analytical Querying for Insights
- Interactive Tableau Dashboards
- Crime Trend & Hotspot Analysis

# ⚙️ Workflow

## 1️⃣ Raw Data Collection
Collected NCRB metro crime datasets in Excel format.

## 2️⃣ Data Cleaning & Transformation
Performed:
- Missing value handling
- Duplicate removal
- Category standardization
- Column normalization
- Dataset merging

Using:
- Python
- Pandas
- Jupyter Notebook

## 3️⃣ Data Warehouse Creation
Created a centralized SQL warehouse for:
- Crime domains
- City-wise analysis
- National trends
- Crime categories

## 4️⃣ Dashboard Development
Built interactive Tableau dashboards featuring:
- KPI Cards
- Trend Analysis
- Donut Charts
- Heat Maps
- Crime Hotspot Analysis
- Dynamic Filters

# 📈 Key Insights

- Delhi emerged as the most dangerous metro city.
- Cybercrime showed the fastest growth trend.
- Crimes against women contributed the largest share.
- Crime cases peaked significantly during 2023.
- Metro cities showed higher crime concentration.
